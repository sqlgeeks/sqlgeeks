1. What are SCALAR Functions
  • A SCALAR function is a user-defined function that takes in one or more parameters and returns a single value each time it is invoked.
  • LEN(): It is used to return the length of a string/text. The LEN function does not count the trailing spaces while it counts the leading spaces at the start of the string/text. It returns NULL for an empty strings/text.

    • SELECT LENGTH(FIRSTNAME) FROM EMPLOYEE

  • ROUND(): It is used to round a number to certain places of decimal.

    • SELECT ROUND(187.499,0)
    OUTPUT:187

    SELECT ROUND(187.599,0)
    OUTPUT:188

    SELECT ROUND(187.599,1)
    OUTPUT:187.6

    SELECT ROUND(187.586,2)
    OUTPUT:187.59

    SELECT ROUND(187.599,-1)
    OUTPUT:190

    SELECT ROUND(187.599,-2)
    OUTPUT:200

    SELECT ROUND(187.599,-3)
    OUTPUT:0

    SELECT ROUND(687.599,-3)
    OUTPUT:1000

  • UCASE(): It is used to convert all alphabetic characters in a input text/string to UPPER-CASE.

    • SELECT UCASE(EMPNAME) FROM EMPLOYEE;

  • LCASE(): It is used to convert all alphabetic characters in a input text/string to LOWER-CASE.

    • SELECT LCASE(EMPNAME) FROM EMPLOYEE;

  • SUBSTRING(): It is used to output the substring from the main string.

    • SELECT SUBSTRING(EMPNAME,1,5) FROM EMPLOYEE

    It will output the substring from the 1st character (included) to the 5th character of the string

    SELECT SUBSTRING(EMPNAME,2) FROM EMPLOYEE

    It will output the string from the 2nd character (included) till the last character of the string

  • NOW(): It is used to return the current data and time in '"YYYY-MM-DD HH:MM:SS"' format.

    • SELECT NOW()
    OUTPUT: 2020-12-19 18:33:17
