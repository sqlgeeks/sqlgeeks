1. What is LIKE operator in SQL ?

- A LIKE operator checks whether a specified charater set/string matches a specified pattern. It uses two wildcards ('%' and '_') with itself, where the % represents 0 or more character while an underscore ('_') represents a single character. Examples -
a. Find the Names of Employees starting with "H"

SELECT NAME FROM EMPLOYEE WHERE NAME LIKE 'H%'
b. Find the Names of Employees ending with "H"
SELECT NAME FROM EMPLOYEE WHERE NAME LIKE '%H'
c. Find the Names of Employees that have "H" in their name
SELECT NAME FROM EMPLOYEE WHERE NAME LIKE '%H%'
d. Find the Names of Employees that have "H" as the second character
SELECT NAME FROM EMPLOYEE WHERE NAME LIKE '_H%'
e. Find the Names of Employees starting with "H" and is of 4 characters length
SELECT NAME FROM EMPLOYEE WHERE NAME LIKE 'H___'
(The 'H' and 3 underscores makes it of length 4)
f. Find the Names of Employees that starts with "H" and ends with "T"
SELECT NAME FROM EMPLOYEE WHERE NAME LIKE 'H%T'
